- Update or delete series by ID
- View a formatted report of all series

## 🧪 How to Run

Make sure you have **Java 17 or later** installed.

### Compile:
```
mvn clean install
mvn compile
```
### Run the App:
```
mvn exec:java
```
### Run the tests:
```
mvn test
```
