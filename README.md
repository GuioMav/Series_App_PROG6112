## 🧰 Tech Stack
- Java 17
- Apache Maven
- JUnit (for testing)

---

## 🛠 Features
- Capture a new series
- Update or delete series by ID
- View a formatted report of all series
- Input validation for age and episode count

---

## 🧪 How to Run

Make sure you have **Java 17 or later** installed.

### Compile:
```
mvn clean install
mvn compile
```
### Run the App:
```
mvn exec:java
```
### Run the tests:
```
mvn test
```
